## How to Report

**Found a violation? Report it immediately.**

1. **In-App Reporting**
   - Open user's profile
   - Tap "Report User"
   - Select reason and provide details

2. **Direct Report**
   - Email: report@edulove.com
   - Include username and details

3. **Emergency**
   - For threats of violence: Contact law enforcement
   - Include EduLove account info in report

## Special Protections

### 18+ Verification
- All users must be 18+
- Age verification is mandatory
- Underage users will be immediately banned

### Student-Only Community
- University verification required
- Protects against fake profiles
- Creates safe student-focused environment

### Consent & Boundaries
- No photos without permission
- Respect "not interested" responses
- No unsolicited explicit messages
- Block/mute available for all users
